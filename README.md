## Installation

### Node

If youâ€™re using [NPM](https://npmjs.com/) in your project, you can add `ipsp-js-sdk` dependency to `package.json`
with following command:

```cmd
npm i --save gulp2go
```

or add dependency manually:

```json
{
  "dependency": {
    "gulp2go":"^1.0"
  }
}
```
